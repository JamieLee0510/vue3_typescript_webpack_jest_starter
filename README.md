# Vue Project Template

## Base Dependencies

-   Vue3
-   Webpack 5
-   Typescript
-   Jest for unit test
-   Eslint config

## Useage

### Project setup

`npm install`

### Compiles and hot-reloads for development

`npm run dev`

### Compiles and minifies for production

`npm run build`

### Run your unit tests

`npm run test:unit`

### Lint the project

just on save
